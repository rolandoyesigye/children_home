# Wings of Tasha Children's Home Website

A static website for Wings of Tasha Children's Home, built with HTML and Tailwind CSS.

## Overview

This website presents the mission, programs, and impact of Wings of Tasha Children's Home in Rubaare, Ntungamo District, Uganda. The site is designed to inform visitors about the organization, showcase their work with vulnerable children, and provide ways for people to get involved through donations, volunteering, and partnerships.

## Pages

- **Home** (`index.html`): Overview of the organization with hero banner, mission, impact numbers, programs preview, stories, and calls to action
- **About Us** (`about.html`): Mission, vision, history, team profiles, governance, and legal information
- **Our Programs** (`programs.html`): Detailed information about the six core programs (Shelter, Health, Education, Psychosocial Support, Child Protection, Vocational Training)
- **Get Involved** (`get-involved.html`): Donation options, child sponsorship, volunteering, partnerships, and in-kind donations
- **News & Stories** (`news.html`): Blog updates, children's stories, photo gallery, and press information
- **Contact** (`contact.html`): Contact form, key personnel, location, and social media links

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Responsive design for mobile and desktop

## Getting Started

1. Clone or download the project files
2. Open any HTML file in a web browser to view the site
3. No build process required - the site uses Tailwind CSS via CDN

## Customization

- Replace placeholder images in the `images/` folder with actual photos
- Update contact information, team details, and content as needed
- Modify colors and styling by adjusting Tailwind classes
- Add actual form processing for the contact form

## Browser Support

The site is designed to work in all modern browsers and includes responsive design for mobile devices.
